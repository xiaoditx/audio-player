# Yumo Audio

## 项目未完工！！！

Yumo Audio是利用WIN32 API实现的C++音频播放库，主要聚焦于音频混合等基础API无法满足的功能，属于自用型库，完全服务于[keyBonk](https://github.com/xiaoditx/keyBonk)所以有一些通用功能将不予考虑

由于上文提到的自用特性，本库结构异常混乱，不支持CMake构建，部分文件缺失需要手动补齐，因此暂不建议直接使用。如果必须使用，请自行改造本库。

本库90%的部分为vibe coding产物，其中的80%未经人工审查，请谨慎使用。

## 一. 兼容性

### 1.平台
Windows独占

### 2.文件格式

（格式按扩展名记）

| 格式 | 是否支持 | 支持程度 |
|:------:|:---------:|:----------:|
|WAV   |支持     |暂不支持携带扩展信息的WAV|
|PCM   |即将支持  | -       |
|MP3   |不支持    | -       |
|MP4   |不支持    | -       |
|M4A   |不支持    | -       |
|OGG   |不支持    | -       |
|FLAC  |不支持    | -       |
|WMA   |不支持    | -       |

### 3.是否兼容C语言

Yumo Audio大量采用C++特性编写，不兼容C语言，虽然可以通过函数包装链接到C，但库内不提供这些包装

## 二. 技术栈

- C++
- WIN32 API
